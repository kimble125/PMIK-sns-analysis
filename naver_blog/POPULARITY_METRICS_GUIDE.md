# 인기 게시물 평가 기준 가이드

## 개요

단순 1:1:1 비율이 아닌, **참여도 가중치 기반 인기도 점수**를 사용하여 진정한 인기 게시물을 식별합니다.

---

## 1. 인기도 점수 산정 방식

### 1.1 기본 개념

**문제점: 단순 합산의 한계**
```
❌ 잘못된 방식: Score = 조회수 + 공감 + 댓글

예시:
게시물 A: 조회 10,000 + 공감 10 + 댓글 5 = 10,015점
게시물 B: 조회 1,000 + 공감 100 + 댓글 50 = 1,150점

→ 게시물 A가 더 높은 점수를 받지만, 실제로는 B가 더 높은 참여도
```

**해결책: 가중치 기반 점수**
```
✅ 올바른 방식: Score = w1×조회수 + w2×공감 + w3×댓글

여기서 w3 > w2 > w1 (댓글이 가장 높은 가치)
```

---

### 1.2 가중치 설정 근거

| 지표 | 가중치 | 근거 |
|------|--------|------|
| **조회수** | 1.0 (기준) | • 수동적 행동 (단순 노출)<br>• 우연히 클릭했을 가능성<br>• 참여 의도 없음 |
| **공감** | 5.0 | • 중간 수준의 참여<br>• 클릭 한 번 필요<br>• 긍정적 반응 표현<br>• 조회수보다 **5배 높은 가치** |
| **댓글** | 10.0 | • 가장 적극적인 참여<br>• 시간과 노력 필요<br>• 깊은 관심 표현<br>• 조회수보다 **10배 높은 가치** |

---

### 1.3 수식

#### A. 원시 점수 (Raw Score)

```
Raw Score = (1.0 × 조회수) + (5.0 × 공감) + (10.0 × 댓글)
```

#### B. 정규화 점수 (0-100 스케일)

```
Normalized Score = 100 × (Raw Score - Min) / (Max - Min)
```

**정규화 이유:**
- 서로 다른 규모의 게시물 비교 가능
- 0-100 범위로 직관적 해석
- 상대적 순위 명확화

## 2. 인기도 기준 종류

### 2.1 가중치 기반 인기도 점수 (권장 ⭐)

**공식:**
```python
popularity_score = 1.0×view + 5.0×like + 10.0×comment
# 정규화 후 0-100 스케일
```

**특징:**
- ✅ 참여도를 정확히 반영
- ✅ 댓글/공감의 가치 인정
- ✅ 조회수 편향 방지

**적용 사례:**
- 전체 인기 게시물 순위
- 판매원 성과 평가
- 콘텐츠 전략 수립

---

### 2.2 참여율 (Engagement Rate)

**공식:**
```python
engagement_rate = (공감 + 댓글) / 조회수 × 100
```

**특징:**
- ✅ 조회수 대비 참여도 측정
- ✅ 소규모 게시물도 높은 점수 가능
- ⚠️ 조회수가 적으면 왜곡 가능

**해석:**
- 1% 미만: 낮은 참여도
- 1-3%: 보통 참여도
- 3-5%: 높은 참여도
- 5% 이상: 매우 높은 참여도

**적용 사례:**
- 콘텐츠 품질 평가
- 타겟 오디언스 적합성 분석

---

### 2.3 댓글률 (Comment Rate)

**공식:**
```python
comment_rate = 댓글 / 조회수 × 100
```

**특징:**
- ✅ 깊은 참여도 측정
- ✅ 커뮤니티 활성도 지표

**해석:**
- 0.1% 미만: 낮음
- 0.1-0.5%: 보통
- 0.5-1%: 높음
- 1% 이상: 매우 높음

---

### 2.4 공감률 (Like Rate)

**공식:**
```python
like_rate = 공감 / 조회수 × 100
```

**특징:**
- ✅ 콘텐츠 호감도 측정
- ⚠️ 클릭 한 번이므로 참여도는 중간

**해석:**
- 1% 미만: 낮음
- 1-3%: 보통
- 3-5%: 높음
- 5% 이상: 매우 높음

## 4. 실전 예시

### 4.1 비교 분석

**게시물 A:**
- 조회수: 10,000
- 공감: 50
- 댓글: 10

**게시물 B:**
- 조회수: 2,000
- 공감: 100
- 댓글: 50

#### 단순 합산 (잘못된 방식)
```
A: 10,000 + 50 + 10 = 10,060점
B: 2,000 + 100 + 50 = 2,150점
→ A가 더 높음 (잘못됨!)
```

#### 가중치 기반 (올바른 방식)
```
A: 1.0×10,000 + 5.0×50 + 10.0×10 = 10,350점
B: 1.0×2,000 + 5.0×100 + 10.0×50 = 3,000점
→ 여전히 A가 높지만, 격차 감소
```

#### 참여율 기준
```
A: (50 + 10) / 10,000 × 100 = 0.6%
B: (100 + 50) / 2,000 × 100 = 7.5%
→ B가 훨씬 높음! (참여도 우수)
```

**결론:**
- **전체 영향력**: A가 우수 (조회수 높음)
- **참여 품질**: B가 우수 (참여율 높음)
- **목적에 따라 선택**

## 10. FAQ

### Q1: 왜 댓글이 공감보다 높은 가중치를 받나요?

**A:** 댓글은 시간과 노력이 필요한 적극적 참여입니다. 공감은 클릭 한 번이지만, 댓글은 생각을 정리하고 입력해야 하므로 더 높은 가치를 가집니다.

### Q2: 조회수가 많은데 인기도 점수가 낮은 이유는?

**A:** 조회수만 많고 공감/댓글이 적으면 참여도가 낮아 점수가 낮을 수 있습니다. 이는 콘텐츠가 타겟 오디언스와 맞지 않거나 품질이 낮을 수 있음을 의미합니다.

### Q3: 어떤 기준을 사용해야 하나요?

**A:** 목적에 따라 다릅니다:
- 전체 영향력: 인기도 점수
- 콘텐츠 품질: 참여율
- 판매원 평가: 인기도 점수 + 참여율 조합

### Q4: 가중치를 어떻게 조정하나요?

**A:** `popularity_analyzer.py`의 `POPULARITY_WEIGHTS` 변수를 수정하거나, 프리셋을 선택하세요. 실험을 통해 최적 가중치를 찾는 것을 권장합니다.
