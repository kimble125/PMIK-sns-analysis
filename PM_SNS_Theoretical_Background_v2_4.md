# **I. 연구 개요** {#i.-연구-개요}

## **1. 연구 배경 및 필요성** {#1.-연구-배경-및-필요성}

### 1.1 네트워크 마케팅 산업의 디지털 전환 {#1.1-네트워크-마케팅-산업의-디지털-전환}

**네트워크 마케팅(Network Marketing)**은 전통적으로 대면 접촉과 개인 네트워크를 기반으로 성장해왔다. 그러나 2010년대 후반부터 SNS 플랫폼의 급격한 확산은 이 산업에 근본적인 변화를 가져왔다. 특히 2020년 COVID-19 팬데믹 이후 비대면 소통이 일상화되면서, SNS를 활용한 제품 홍보와 판매원 모집이 네트워크 마케팅의 핵심 전략으로 자리잡았다.

**산업 트렌드:**[^1][^2]

- **글로벌 직접판매 시장 규모**: 
  - 2022년: $186.1B → 2023년: $193.4B (+3.9%)
  - 2024년: $201.2B (추정)
  
- **한국 직접판매 시장**:
  - 2022년: 약 4조 6천억 원 → 2023년: 약 4조 8천억 원 (+4.3%)
  - 2024년: 약 5조 2천억 원 (추정, +8.3%)
  
- **디지털 채널 비중**:
  - 2020년: 35% → 2022년: 52% → 2024년: 68% (추정, +33%p)
  - 주요 채널: SNS(43%), 이커머스(25%), 모바일 앱(18%), 기타(14%)

[^1]: Direct Selling News. (2024). *Global 100 List 2024*. Retrieved from https://www.directsellingnews.com/
[^2]: 한국직접판매산업협회. (2024). *2023년 직접판매 산업 통계보고서*. Retrieved from http://www.kdsaa.or.kr/

### 1.2 PM-International의 한국 시장 현황 {#1.2-pm-international의-한국-시장-현황}

PM-International AG는 1993년 독일에서 설립된 건강기능식품 전문 네트워크 마케팅 기업이다. 한국 법인인 PM-International Korea(이하 PMI코리아)는 2018년 설립 이후 빠르게 성장하여, **2024년 기준 한국 네트워크 마케팅 시장 3위**를 차지하고 있다.[^3] 

더욱 주목할 만한 점은 PMI코리아가 **한국 네트워크 마케팅 회사들 중 건강기능식품 수입 1위**를 기록하고 있다는 것이다.[^4] 이는 PMI코리아의 제품 경쟁력과 판매 네트워크의 효율성을 입증한다.

[^3]: 매경비즈. (2024). "네트워크 마케팅 업계 '빅3' 격돌". Retrieved from https://www.mknews.kr/?mid=view&no=42773
[^4]: 매일마케팅신문. (2024). "피엠인터내셔널 코리아, 건강기능식품 수입 1위 달성". Retrieved from https://www.maeilmarketing.com/news/articleView.html?idxno=10691

**PMI코리아의 특징:**[^5]

- **디지털 우선 전략**: 설립 초기부터 모바일 앱과 SNS 중심의 마케팅 채택
- **모바일 앱 통합**: 주문, 교육, 커뮤니케이션을 하나의 앱(PM-International App)에 통합
- **SNS 친화적 문화**: 팀파트너(판매원)의 자율적 SNS 활동을 장려하며 콘텐츠 가이드라인 제공

[^5]: PM-International Korea. (2024). *팀파트너 비즈니스 가이드북*. 내부 자료.

**2025년 디지털 커뮤니케이션 강화 전략:**[^6]

PMI코리아는 2025년 마케팅 전략의 3대 핵심 방침으로 다음을 발표했다:
1. **디지털 커뮤니케이션** 강화
2. **브랜드 매니지먼트 관리** 체계화
3. **제품 포트폴리오** 다각화

특히 마케팅본부 탁창준 본부장은 "틱톡 글로벌 이용자 수 13억명, 유튜브 숏츠 일 평균 조회수 300억회, 릴스 일 평균 재생 수 1400억회 등 다양한 숏폼 콘텐츠들이 인기를 얻고 있다"며 "올해는 획기적인 숏폼 제작으로 회사를 알리는데 힘쓰겠다"고 밝혔다. 이는 **본 프로젝트가 PMI코리아의 핵심 전략 방향과 직접적으로 연계**되어 있음을 시사한다.

[^6]: 매일마케팅신문. (2024). "피엠인터내셔널 코리아, 2025년 디지털 커뮤니케이션 강화". Retrieved from https://www.maeilmarketing.com/news/articleView.html?idxno=10955

그러나 이러한 성장 이면에는 **체계적인 데이터 분석의 부재**라는 과제가 존재한다. 현재 PMI코리아는 약 15,000-20,000명의 팀파트너가 활동 중이며, 이들의 SNS 활동이 실제 매출과 조직 확장에 어떤 영향을 미치는지를 정량적으로 파악할 필요가 있다.

### 1.3 연구의 필요성 {#1.3-연구의-필요성}

**1) 기업 관점:**

- 고성과 팀파트너의 SNS 전략 패턴 발견 및 확산
- 저성과 그룹 대상 맞춤형 교육·상담 콘텐츠 개발
- 브랜드 리스크(허위·과장 광고) 조기 감지 시스템 구축
- 데이터 기반 마케팅 의사결정 지원

**2) 판매원 관점:**

- 데이터 기반 SNS 마케팅 가이드라인 제공받음
- 플랫폼별 최적화된 콘텐츠 전략 제시받음
- 성공 사례 벤치마킹 기회 획득

**3) 학술 관점:**

- 한국형 네트워크 마케팅 디지털 전환 연구
- 40-60대 여성의 SNS 소비·생산 행동 분석
- 직접판매 산업의 플랫폼 경제 적응 모델

## 2. 연구 목적 및 범위 {#2.-연구-목적-및-범위}

### 2.1 연구 질문 (Research Questions) {#2.1-연구-질문-(research-questions)}

**핵심 질문:**

"PMI코리아 팀파트너의 SNS 활동 패턴은 무엇이며, 이것이 판매 성과 및 조직 확장에 어떤 영향을 미치는가?"

**세부 질문:**

1. 5개 주요 SNS 플랫폼(네이버 블로그, 유튜브, 인스타그램, 카카오스토리, 페이스북)에서 팀파트너의 **활동 빈도**, **콘텐츠 유형**, **반응도(engagement)** 등은 어떠한가?
   - *반응도: 좋아요, 댓글, 공유 등 사용자 상호작용 지표*
   
2. 고성과 팀파트너와 저성과 팀파트너의 SNS 활동에는 어떤 차이가 있는가?

3. 특정 플랫폼 또는 콘텐츠 유형이 매출 증대나 신규 팀파트너 모집에 더 효과적인가?

4. 브랜드 리스크를 초래할 수 있는 부적절한 콘텐츠(허위·과장 광고)의 확산 양태는?

### 2.2 연구 범위 {#2.2-연구-범위}

**시간적 범위:**

- **데이터 수집 기간**: 2025년 1월 ~ 2025년 3월 (3개월)
- **분석 대상 기간**: 2024년 7월 ~ 2024년 12월 (최근 6개월 게시물)

**공간적 범위:**

# III. 네트워크 마케팅과 SNS의 결합 {#iii.-네트워크-마케팅과-sns의-결합}

## 1. 네트워크 마케팅의 기본 메커니즘 {#1.-네트워크-마케팅의-기본-메커니즘}

### 1.1 구전(Word-of-Mouth) 기반 확산 {#1.1-구전(word-of-mouth)-기반-확산}

네트워크 마케팅은 **신뢰 기반 구전(Word-of-Mouth)**을 핵심 메커니즘으로 한다. 제품을 경험한 만족 고객이 주변 지인에게 추천하고, 추천받은 사람이 다시 판매원으로 참여하면서 조직이 기하급수적으로 확장된다.

**핵심 요소:**

1. **신뢰 자본**: 친구·가족 등 기존 관계망의 신뢰를 기반으로 제품 추천
2. **경험 공유**: 직접 사용 후기와 변화 사례 공유
3. **동기부여**: 하위 조직 매출에 연동된 보상 시스템

### 1.2 전통적 한계 {#1.2-전통적-한계}

**물리적 제약:**

- 대면 미팅 필요: 시간·공간 제약
- 제한된 인맥: 개인이 직접 접근 가능한 네트워크 크기 한정
- 고비용 구조: 제품 시연, 세미나 개최 등 고정비용 발생

**확장성 한계:**

- 1명이 관리 가능한 조직 규모 제한
- 지역적 편중 (도심 vs. 지방)

## 2. SNS 플랫폼이 가져온 구조적 변화 {#2.-sns-플랫폼이-가져온-구조적-변화}

### 2.1 패러다임 전환 {#2.1-패러다임-전환}

| 전통적 방식 | SNS 시대 |
|:---|:---|
| 대면 미팅 중심 | 온라인 콘텐츠 중심 |
| 1:1 또는 소규모 세미나 | 1:N 대중 전파 (바이럴 가능) |
| 물리적 인맥 의존 | 온라인 네트워크 확장 |
| 시간·공간 제약 | 24/7 접근 가능 |
| 고비용 (장소, 교통비) | 저비용 (콘텐츠 제작비) |

### 2.2 구체적 변화 양상 {#2.2-구체적-변화-양상}

**1) 콘텐츠화:**

- **Before**: "이 제품 정말 좋아요" (말로만)
- **After**: 사용 전후 사진, 리뷰 영상, 일상 브이로그 등 **시각적 증거** 제공

**2) 바이럴 효과:**

- 한 게시물이 수천~수만 명에게 도달 가능
- 해시태그, 공유 기능으로 확산 가속

**3) 타겟팅:**

- 연령대, 관심사별 맞춤 콘텐츠 제작

**SNS 방식:**

## 2. 기술적 구현 방안 {#2.-기술적-구현-방안}

**주요 단계 요약:**

- Step 1: YouTube Data API v3로 영상 메타데이터 수집
- Step 2: YouTube Transcript API로 자막 추출
- Step 3: Whisper(OpenAI)로 음성-텍스트 변환 (자막 없는 경우)

**인스타그램:**
- Step 1: Instagram Graph API로 Business Account 데이터 수집
- Step 2: 캡션, 해시태그, 반응 지표 추출
- Step 3: 이미지 OCR 처리

**데이터베이스:**
- Azure SQL Database 사용
- 테이블: posts, engagement_metrics, post_metadata, author_performance

**자동화:**
- Azure Data Factory로 일일 수집 파이프라인 구축
- 스케줄링: 매일 새벽 2시 자동 실행

## 3. 데이터 분석 기법 {#3.-데이터-분석-기법}

*자세한 분석 방법은 '인턴 업무 계획서 v2.2'에 상세히 기술됨.*

### 3.1 클러스터링 (K-Means) {#3.1-클러스터링-(k-means)}

**목적:** 팀파트너를 SNS 활동 패턴에 따라 그룹화

**주요 변수:**
- 주간 게시 빈도
- 평균 반응률 (좋아요+댓글)/조회수
- 플랫폼 다양성 (활동 플랫폼 수)
- 콘텐츠 길이

**예상 군집:**
- Cluster 0: "고빈도·고반응형"
- Cluster 1: "중빈도·안정형"
- Cluster 2: "저빈도·캐주얼형"
- Cluster 3: "비활동형"

### 3.2 자연어 처리 (NLP) {#3.2-자연어-처리-(nlp)}

**목적:** 게시물 텍스트 분석을 통한 군집화 및 주제 분석

**주요 기법:**
- **텍스트 벡터화**: TF-IDF, Word2Vec 등을 활용하여 텍스트를 수치 벡터로 변환
- **토픽 모델링**: LDA(Latent Dirichlet Allocation)를 통한 주요 주제 추출
- **키워드 추출**: 빈도 분석 및 워드클라우드 생성

*참고: 감성 분석은 선택적으로 적용. 판매 홍보 콘텐츠의 특성상 대부분 긍정적이므로 변별력이 낮을 수 있음. 다만 댓글 분석 시에는 유용할 수 있어 필요시 KoBERT 등 활용 가능.*

### 3.3 회귀 분석 (Regression) {#3.3-회귀-분석-(regression)}

**목적:** SNS 활동이 성과(매출, 리크루팅)에 미치는 영향 정량화

**모델:** OLS 회귀분석

**종속변수:**
- 월 매출
- 월 신규 리크루팅 수

**독립변수:**
- 주간 게시 빈도
- 평균 반응률
- 팔로워 수
- 활동 플랫폼 수
